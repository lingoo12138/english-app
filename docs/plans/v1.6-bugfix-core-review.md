## 修法

### Bug #1+#2: WritePage useEffect 切 tab

**原代码** (line 75-83):
```tsx
useEffect(() => {
  if (activeTab === 'history') {
    loadHistory()
  } else {
    setInput('')
    setResult(null)
    setError('')
    setAddedWords(new Set())
  }
}, [activeTab])
```

**修法**: 只在切到 history 时 loadHistory, 切回 write 时**保留** input/result/addedWords
- loadHistory 不重置 write tab state
- handleHistoryItem 在 useEffect 之前 setInput/setResult, useEffect 不会覆盖 (因为 activeTab 没变)

```tsx
useEffect(() => {
  if (activeTab === 'history') {
    loadHistory()
  }
}, [activeTab])
```

### Bug #3+#4: ListenPage DictationMode/QuestionsMode 切 lesson

**原代码** (DictationMode line 232-236):
```tsx
const [answers, setAnswers] = useState<Record<number, string>>({})
const [submitted, setSubmitted] = useState(false)
const [playing, setPlaying] = useState(false)
const [addedWords, setAddedWords] = useState<Set<string>>(new Set())
```

**修法**: 加 `useEffect [lesson]` 重置状态
```tsx
useEffect(() => {
  setAnswers({})
  setSubmitted(false)
  setAddedWords(new Set())
}, [lesson])
```

### Bug #5+#6: ErrorExplainButton/UsageButton setLoading(true) 缺失

const handleClick = async () => {

### Bug #7: AIChat STT 累积无 MAX_LEN 限制

**原代码** (line 200-203):
```tsx
if (isFinal) {
  setInput(prev => (prev ? prev + ' ' : '') + text)
  setSttInterim('')
}
```

**修法**: 加 MAX_LEN = 500, 累积超过截断
```tsx
const MAX_INPUT = 500
if (isFinal) {
  setInput(prev => {
    const next = (prev ? prev + ' ' : '') + text
    return next.length > MAX_INPUT ? next.slice(0, MAX_INPUT) : next
  })
  setSttInterim('')
}
```

### Bug #8: WritePage 截断逻辑

**原代码** (line 90-94):
```tsx
if (input.length > MAX_LEN) {
  setError(`文本超过 ${MAX_LEN} 字符(当前 ${input.length}),已截断`)
  setInput(input.slice(0, MAX_LEN))
}
```

**修法**: 用 text 变量, 重新 send
```tsx
if (input.length > MAX_LEN) {
  setError(`文本超过 ${MAX_LEN} 字符(当前 ${input.length}),已截断`)
  const truncated = input.slice(0, MAX_LEN)
  setInput(truncated)
  // 后续 text 变量使用 truncated
}
const text = input.length > MAX_LEN ? input.slice(0, MAX_LEN) : input
```
