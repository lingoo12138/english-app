## ⚠️ 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| LLM 成本 | 用户每错一题调 1 次 | 缓存 + 限流 |
| LLM 响应慢 (10s) | 用户等待 | loading + 缓存命中秒出 |
| JSON 解析失败 | 显示空 | try-catch + fallback 简单文本 |
| 缓存击穿 (同一错词高频) | 重复调 | 内存 + IndexedDB 双层缓存 |
| 用户无 LLM 渠道 | 无法讲解 | Mock 渠道 (固定解释) |
| IndexedDB schema 升级 | 旧数据迁移 | v2 schema 兼容 v1 |
