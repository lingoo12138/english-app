## 🎯 下一步(按优先级)

### 短期(P3,1-2 周)

1. **📱 微信小程序** —— uni-app 编译,把 Web 端代码复用过去
   - 小程序原生:单词卡 + TTS + IndexedDB
   - 不支持的功能回退:Web Speech API → 小程序同声传译
2. **📚 例句扩充** —— 把每个高频词从 1.8 句提到 3-5 句
   - 数据源:ECDICT 100万词条 / Tatoeba 100万句对 / 手动挑高质量
3. **🧹 P2 微优化** —— 47 个 P2(代码风格/微优化)
   - 性价比低,非阻塞

### 中期(P4-P5,2-4 周)

1. **🤖 Android App** —— uni-app 编译 APK
2. **📷 图片识别** —— 拍照识物,返回英文 + 发音 + 例句
3. **🌱 词根词缀扩充** —— 覆盖率从 54% → 80%+
4. **👂 听力模式** —— 短文 + 听写

### 远期(需要后端)

1. **☁️ 账号系统 + 多端同步** —— 用户数据跨设备
2. **🤖 AI 对话陪练** —— LLM API
3. **📊 学习社区** —— 分享错题本/笔记
