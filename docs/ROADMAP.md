## 🎯 下一步(按优先级)

### 短期(P2,1-2 周)
1. **🎤 跟读评测**(3-5 天) — MediaRecorder + 简单评分
   - 跟读单词,录音对比原音
   - 跟读例句,跟读得分
2. **🎬 场景专题课**(7-10 天) — 5-8 个场景
   - 餐厅/问路/办公/求职/差旅...
   - 5-10 句 + 真人对话视频 + 跟读 + 小测
3. **📷 图片识别**(7+ 天) — 拍照识物
   - 识别物体 → 返回英文 + 发音 + 例句
   - 接入百度/腾讯图像识别 API

### 中期(P3,2-4 周)
1. **📱 微信小程序** — uni-app 编译
2. **🤖 Android App** — uni-app 编译 APK
3. **🎨 例句扩充** — 核心高频词补到 3-5 句
4. **🌱 词根词缀扩充** — 覆盖率从 54% → 80%+

### 远期(需要后端)
1. **☁️ 账号系统 + 多端同步**
2. **🤖 AI 对话陪练** — LLM API
3. **👂 听力模式** — 短文 + 听写
4. **📊 学习社区** — 分享错题本/笔记
